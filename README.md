## Anki Installation and Deck Import Guide

### Step 1: Installing Anki
1. Download Anki from the official website: [https://apps.ankiweb.net/](https://apps.ankiweb.net/)
2. Install the application following your system's instructions

### Step 2: Importing the Teochew Deck
#### Download the deck:
- Direct download: [Teochew.apkg](https://github.com/Stephane90/ankiTC/raw/refs/heads/main/Teochew.apkg)

#### Import instructions:
1. Open the downloaded `Teochew.apkg` file
2. Anki will automatically:
   - Create a new deck (if it doesn't exist)
   - Import all cards
